## Questions We Love

Hermit is a kernel project with unique concepts. We especially welcome questions about:

- **Governance model** -- How approvals, receipts, and proofs work
- **Policy configuration** -- Choosing the right policy profile for your use case
- **Plugin development** -- Building adapters, hooks, and tools
- **Self-iteration** -- Using the governed self-improvement pipeline
- **MCP integration** -- Connecting Hermit to supervisor agents

## Before You Ask

1. Check the [FAQ](docs/faq.md)
2. Search [existing issues](https://github.com/heggria/Hermit/issues) and [discussions](https://github.com/heggria/Hermit/discussions)
3. Include your Hermit version (`hermit --version`), Python version, and OS when reporting issues
