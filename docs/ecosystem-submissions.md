## Submission Guidelines

- Only submit after Hermit has a stable README, docs site, and working examples
- Each submission should highlight the unique governance/receipts/proof angle
- Link to the docs site (https://heggria.github.io/Hermit/) as primary reference
- Be honest about alpha status

## Status Key

- **Pending**: Not yet submitted
- **Submitted**: PR or issue opened
- **Accepted**: Merged or listed
