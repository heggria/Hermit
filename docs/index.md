# Hermit

![Hermit macOS menu bar icon](assets/hermit-macos-icon.svg){ width="96" }

Hermit turns agent work into governed, inspectable, local-first execution.

Important AI work becomes durable tasks with approvals, scoped authority, receipts, proof summaries,
and rollback-aware recovery.

## Why Contributors Care

- Task-first kernel instead of disposable chat state
- Policy, approval, and scoped authority around side effects
- Receipts, proof bundles, and rollback-aware recovery paths
- Operator-facing docs that separate current state from target state
